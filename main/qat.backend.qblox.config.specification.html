class ConnectionConfig(bulk_value=<factory>, out0=None, out1=None, out2=None, out3=None, out4=None, out5=None, acq_I=None, acq_Q=None)

Bases: object

Configuration for the sequencer’s connection to the analogue input/output paths.

Parameters:

  • bulk_value¶ (list[str]) – A list of strings in the format <direction><channel> or <direction><I-channel>_<Q-channel>: <direction> must be ‘in’ to make a connection between an input and the acquisition path, ‘out’ to make a connection from the waveform generator to an output, or ‘io’ to do both. <channel> must be integer channel indices. If only one channel is specified,the sequencer operates in real mode; if two channels are specified, it operates in complex mode.

  • out0¶ (Optional[str]) – Component config of a sequencer’s connection to output 0, if any. Possible values are ‘I’, ‘Q’, ‘IQ’, or ‘off’

  • out1¶ (Optional[str]) – Component config of a sequencer’s connection to output 1, if any. Possible values are ‘I’, ‘Q’, ‘IQ’, or ‘off’

  • out2¶ (Optional[str]) – Component config of a sequencer’s connection to output 2, if any. Possible values are ‘I’, ‘Q’, ‘IQ’, or ‘off’

  • out3¶ (Optional[str]) – Component config of a sequencer’s connection to output 3, if any. Possible values are ‘I’, ‘Q’, ‘IQ’, or ‘off’

  • out4¶ (Optional[str]) – Component config of a sequencer’s connection to output 4, if any. Possible values are ‘I’, ‘Q’, ‘IQ’, or ‘off’

  • out5¶ (Optional[str]) – Component config of a sequencer’s connection to output 5, if any. Possible values are ‘I’, ‘Q’, ‘IQ’, or ‘off’

  • acq_I¶ (Optional[str]) – Input config for the ‘I’ input of the acquisition path of this sequencer is connected to, if any. Possible values are ‘in0’, ‘in1’, or ‘off’

  • acq_Q¶ (Optional[str]) – Input config for the ‘Q’ input of the acquisition path of this sequencer is connected to, if any. Possible values are ‘in0’, ‘in1’, or ‘off’
