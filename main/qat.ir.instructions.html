class Sweep(**data)

Bases: Instruction

This is a global meta-instruction that performs a sweep over values, effectively performing a loop over the instructions replacing a variable with a specific value per time.

Nested sweeps are run in the order they’re added and are performed after repeats. So a 1000 repeat with a sweep of four values, followed by a sweep with two values will run a total of 8000 iterations.
